# Privacy Policy

**`Last Updated: April 1, 2026`**

:::success
This Privacy Policy explains how **Pandoryx** ("the Bot", "we", "our", or "us") collects, uses, and protects information when you interact with the bot through Discord. By using Pandoryx, you agree to the collection and use of information in accordance with this policy.
:::

<hr className="md-divider-gradient" />

## 2. Information We Collect
Pandoryx collects only the data necessary to provide moderation, automation, and utility features.

### 2.1 User Data
- Discord User ID  
- Username and display name  
- Server (Guild) ID  
- Channel ID  
- Role IDs (for permission systems)

### 2.2 Moderation Data
- Warnings, infractions, and moderation history  
- Punishment records (timeouts, kicks, bans, etc.)  
- Moderator actions and logs  

### 2.3 Content Data
- Messages processed for moderation (e.g., anti-link, filters)  
- Inputs provided through commands  
- Embed content created or stored via the bot  

### 2.4 Configuration Data
- Server settings (automations, permissions, modules enabled/disabled)  
- Ticket system data (panels, tickets, transcripts if enabled)  
- Logging configurations  

### 2.5 System Data
- Command usage logs  
- Error logs for debugging and performance improvements  

<hr className="md-divider-gradient" />

## 3. How We Use Your Information
We use collected data strictly to:

- Provide moderation and server management features  
- Enforce server rules and automate actions  
- Store server configurations and preferences  
- Generate logs and audit trails  
- Improve performance, reliability, and security  

We do **not** sell, rent, or trade your data.

<hr className="md-divider-gradient" />

## 4. Data Storage
All data is stored securely using databases (such as MongoDB or similar systems) and is only accessible to authorized systems and developers maintaining Pandoryx.

We implement reasonable safeguards to protect your data, though no system can be guaranteed completely secure.

<hr className="md-divider-gradient" />

## 5. Data Retention
Data is retained only for as long as necessary to provide functionality.

Server owners and administrators may control certain stored data (such as logs or moderation records), depending on configuration.

<hr className="md-divider-gradient" />

## 6. Third-Party Services
Pandoryx operates within Discord and may rely on third-party services such as:

- Discord API  
- Hosting providers  
- Database services  

These services may process limited data required for operation.
We are not responsible for how Discord or these third parties handle your data.

<hr className="md-divider-gradient" />

## 7. Data Sharing
We do not share your data except in the following cases:

- When required by law  
- To prevent abuse, fraud, or security threats  
- To comply with legal obligations  

<hr className="md-divider-gradient" />

## 8. Your Rights
You have the right to:
- Request access to your stored data  
- Request correction of inaccurate data  
- Request deletion of your data  

Server administrators may also manage or delete certain server-level data.
To make a request, contact the developer or support server associated with Pandoryx.

<hr className="md-divider-gradient" />

## 9. Data Deletion
If the bot is removed from a server, some server data may remain stored unless manually deleted.

You may request full deletion of your personal data at any time.

<hr className="md-divider-gradient" />

## 10. Children's Privacy
Pandoryx is not intended for users under the age of 13.  
If we become aware that data from a user under 13 has been collected, it will be deleted.

<hr className="md-divider-gradient" />

## 11. Changes to This Policy
We may update this Privacy Policy from time to time.
Changes will be reflected by updating the "Last Updated" date at the top of this page.
Continued use of the bot after changes means you accept the updated policy.

<hr className="md-divider-gradient" />
